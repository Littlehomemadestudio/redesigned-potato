## 🚀 نصب و راه‌اندازی

### روش 1: نصب مستقیم

1. **کلون کردن پروژه:**
```bash
git clone https://github.com/yourusername/war-simulation-bot.git
cd war-simulation-bot
```

2. **نصب وابستگی‌ها:**
```bash
pip install -r requirements.txt
```

3. **تنظیم توکن:**
```bash
cp config_example.py config.py
# فایل config.py را ویرایش کرده و توکن ربات خود را قرار دهید
```

4. **اجرای ربات:**
```bash
python run.py
```

### روش 2: استفاده از Make

1. **راه‌اندازی کامل:**
```bash
make setup
```

2. **اجرای ربات:**
```bash
make run
```

3. **اجرای تست‌ها:**
```bash
make test
```

### روش 3: استفاده از Docker

1. **ساخت تصویر:**
```bash
docker build -t war-simulation-bot .
```

2. **اجرای ربات:**
```bash
docker run -d --name war-bot -v $(pwd)/data:/app/data war-simulation-bot
```

3. **استفاده از Docker Compose:**
```bash
docker-compose up -d
```

### تنظیمات پیشرفته

#### فایل config.py
```python
# توکن ربات بله
BOT_TOKEN = "your_bot_token_here"

# تنظیمات بازی
GAME_CONFIG = {
    "RESOURCE_PRODUCTION_INTERVAL": 5,  # دقیقه
    "MAX_COUNTRY_LEVEL": 50,
    "MIN_ATTACK_POWER": 100,
    "CONQUEST_CHANCE_BASE": 10,
}

# تنظیمات لاگ
LOG_CONFIG = {
    "ENABLE_FILE_LOGGING": True,
    "LOG_FILE": "war_logs.txt",
    "LOG_LEVEL": "INFO"
}
```

#### متغیرهای محیطی
```bash
export BOT_TOKEN="your_bot_token_here"
export DATA_FILE="war_data.txt"
export LOG_FILE="war_logs.txt"
```

### سوالات متداول

#### س: چگونه توکن ربات دریافت کنم؟
ج: از [@BotFather](https://t.me/BotFather) در بله استفاده کنید.

#### س: چرا منابع تولید نمی‌شود؟
ج: از دستور `/collect` استفاده کنید. منابع هر ساعت تولید می‌شوند.

#### س: چگونه سیستم مدیریت تغییر دهم؟
ج: از دستور `/change_government [نوع]` استفاده کنید.

#### س: چرا نمی‌توانم به بازیکن حمله کنم؟
ج: ممکن است بازیکن خیلی قوی باشد یا در اتحاد شما باشد.

#### س: چگونه کالکتور بخرم؟
ج: از دستور `/buy_collector [نوع]` استفاده کنید.
